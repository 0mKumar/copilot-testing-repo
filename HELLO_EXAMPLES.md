# Hello World Examples

This directory contains "Hello, World!" implementations in multiple programming languages, demonstrating the classic first program across different ecosystems.

## Available Languages

| Language   | File              | Run Command              |
|------------|-------------------|--------------------------|
| Python     | `hello_world.py`  | `python3 hello_world.py` |
| JavaScript | `hello_world.js`  | `node hello_world.js`    |
| Java       | `HelloWorld.java` | `javac HelloWorld.java && java HelloWorld` |
| Bash       | `hello_world.sh`  | `./hello_world.sh`       |
| C          | `hello_world.c`   | `gcc hello_world.c -o hello_world_c && ./hello_world_c` |

## Running All Tests

To test all implementations at once:

```bash
./test_hello_world.sh
```

This script validates that each implementation outputs the expected "Hello, World!" message.

## Purpose

These examples serve as:
- Basic syntax demonstrations for each language
- Simple programs for testing coding agent capabilities
- Reference implementations for "Hello, World!" programs
- Examples for automated testing and validation
